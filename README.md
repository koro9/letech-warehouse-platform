# LeTech Warehouse Platform

LeTech 仓库管理 Web 平台，覆盖**内部仓库作业**、**3PL 商家自助**、**PO 收貨**三大业务场景。

---

## 📊 Dashboard 系統 — 内部仓库员工

| 模块 | 说明 |
|---|---|
| 控制台 | 当日出库扫描 / 今日 · 明日打印件数 KPI 总览 |
| 出库扫码 | 扫订单号载入待出库清单，扫商品条码逐件核验，可选打印 |
| 面单 | 今日 / 明日面单文件分屏列表，按订单号下载 |
| 拆单 | 多订单合并面单的拆分 |
| 标签 | 商品条码 + 数量直打 |
| 订单查询 | 日期范围 / 订单号 / 状态多条件筛选 |
| 库存对比 | 上传库存文件，与系统数据比对，历史可追溯 |
| 对账 | 上传对账单文件，记录处理时间 |
| 个人信息 / 用户管理 | 账号信息维护，角色 / 权限管理 |

---

## 🏭 3PL 倉庫平台 — 3PL 商家自助

| 模块 | 说明 |
|---|---|
| 智能查詢中心 | 本地资料库（SKU / Barcode / 中英文名）模糊搜索 + DEAR 即时库存查询（On Hand / Allocated / Available） |
| 3PL 貨品檢測 | 选择品牌 → 建立或加入 5 位数任务码 → 扫码进度同步 |
| Yummy 3PL | HKTVmall Yummy Delivery Note PDF 解析 → 重複检测 → 标签打印 |
| Anymall 3PL | Anymall Delivery Note 极速解析 |
| Hello Bear 3PL | Hello Bear Delivery Note 解析 |
| Homey 3PL | Homey Delivery Note 解析（含蟲蟲 / Repack / 普通三种 Label 类型） |

---

## 📦 PO 收貨平台 — 收货员

| 模块 | 说明 |
|---|---|
| PO 點貨 | 输入 PO 载入清单 → 扫条码定位 SKU → 多仓库 / 多效期数量录入；Combo 装自动按倍数计算实拿件数；Remarks 处理需录入操作员姓名留痕 |
| 收貨分配 | 单 PO 多商品的入仓分配：3PL / WS / SD4 三个标准仓 + 最多 2 个自定义额外仓；支持 ×2 / ×3 / ×5 Combo 子行；SD4 自动结余；导出 Excel |
| Transfer Order | PO 下多张 TR 单的揀貨进度追踪；扫码定位品项；BOM 子件自动展开计算；**截單功能**：将一张 TR 拆为"已揀（第一轉）+ 剩餘（第二轉）"两张单据出貨 |
